Admin API Documentation

Admin Routes

Route: /admin/token
Methods: GET
מחזיר את כל הטוקנים והאפליקציות שעובדות
Route: /admin/token
Methods: POST
מוסיף טוקן חדש למערכת עבור אפליקציה מסוימת
Route: /admin/token/<app_id>
Methods: PUT
מעודכן טוקן קיים לפי מזהה אפליקציה
Route: /admin/token/<app_id>
Methods: DELETE
מוחק טוקן קיים לפי מזהה אפליקציה

User Routes

Route: /user
Methods: POST
יוצר משתמש חדש במערכת
Route: /user/<user_id>
Methods: GET
מחזיר פרטי משתמש לפי מזהה
Route: /user/<user_id>
Methods: PUT
מעודכן פרטי משתמש קיים לפי מזהה
Route: /user/<user_id>
Methods: DELETE
מוחק משתמש קיים לפי מזהה
