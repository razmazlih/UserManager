API Documentation

User Routes

Route: /user
Method: POST
Description: Creates a new user in the system
Authentication: Requires Authorization token in the request header
Required Data (JSON):
{"phone": "unique phone number", "name": "user name", "email": "email address"}
Route: /user/<user_id>
Method: GET
Description: Retrieves user details by ID
Authentication: Requires Authorization token in the request header
Required Data:
user_id should be replaced with the user's phone number
Route: /user/<user_id>
Method: PUT
Description: Updates an existing user's details by ID
Authentication: Requires Authorization token in the request header
Required Data (JSON):
{"phone": "unique phone number", "name": "new name", "email": "updated email address"}
Route: /user/<user_id>
Method: DELETE
Description: Deletes an existing user by ID
Authentication: Requires Authorization token in the request header
Required Data:
user_id should be replaced with the user's phone number
